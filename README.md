# CS 300 Portfolio

## What was the problem you were solving in the projects for this course?
The goal of this course was to learn about different data structures and determine which one worked best for storing, searching, and displaying course information.
## How did you approach the problem? 
I compared vectors, hash tables, and binary search trees by analyzing their runtime and memory usage. 
## How did you overcome any roadblocks you encountered while going through the activities or project?
One of the biggest challenges was fixing errors in my code. I tested different solutions and made corrections until the program worked properly.
## How has your work on this project expanded your approach to designing software? 
This project helped me understand how important it is to choose the right data structure for a program.
## How has your work evolved the way you write programs that are maintainable, readable, and adaptable? 
This project taught me skills that will help me maintain, update, and improve my programs. It also helped me learn how to organize my code and use clear variable names, which makes programs easier to read and understand.
